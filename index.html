My First Website - Workshop

Time to build your first website! :)

Download this project as a .zip file Download this project as a tar.gz file

Yes, lets make a website!

Some theory first.

So what is HTML?

Hyper Text Markup Language

Enough with the theory, lets make a website!

Download the zip file and unzip it.

Open index.html and paste the following: 

<!DOCTYPE html>
<html>
  <head>

  </head>
  <body>

  </body>
</html>

To add a title to your website, add <title> My Website </title> in between the head tags

Let's explore some other html tags, these go in between the body tags

Header tag: 

<h1> This is a header </h1>

Paragraph tag: 

<p> This is a paragraph </p>

Image tag: 

<img src="http://i.imgflip.com/ry39.jpg"></img>

Anchor tag:

<a href="http://hacksu.cs.kent.edu">Take me to hacksu!</a>

Div tag:

<div>Something in a div, Brian!</div>

Lets add some style with CSS

We can either setup the style in the head tag in index.html or in style.css.

In index.html

<style>
  div {
    background-color: blue;
  }
</style>

In style.css

div {
  background-color: blue;
}

Lets add some functionality with JavaScript

We can either setup the script in the head tag in index.html or in script.js.

In index.html

<head>
  <script>
    var hack = document.getElementById("hacksu");
    hack.onclick = function() {
      alert("hello!");
    }
  </script>
</head>
<body>
    <div id="hacksu">Click me</div>
</body>

In script.js

var hack = document.getElementById("hacksu");
hack.onclick = function() {
  alert("hello!");
}
